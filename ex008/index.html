Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica)

Nessa frase temos um termo em destaque usando a tag <strong> (não semântica)

Italico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> (não semântica)

Nessa frase temos um termo em ênfase usando a tag <em> (semântica)

Sublinhado / Inserido

Nessa frase temos um termo em sublinhado usando a tag <u> (não semântica)

Nessa frase temos um termo em Inserido usando a tag <ins> (semântica)

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande com a tag <big> e um texto pequeno com a tag <small>

Texto deletado

Podemos marcar um texto excluído com a tag <del> para indicar que ele deve ser lido, mas não considerado

Texto sobrescrito

Para inserir coisas do tipo x20+3 com a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O com a tag <sub>

Resumo

<strong> Destaque
<em> Ênfase
<ins> Inserido
<mark> Marcado
<small> Pequeno
<del> Deletado
<sup> sobrescrito
<sub> subscrito
